## Agenda
* Can Fst be negative? In this [paper (page 330)](https://onlinelibrary.wiley.com/doi/epdf/10.1111/j.1469-1809.1949.tb02451.x), Wright stated that "Fst is necessarily positive but Fis and Fit can be negative". In this [paper (page 400)](https://www.jstor.org/stable/2406450?seq=1) Wright stated that "Correlation coefficients vary between -1 and +1 while probabilities can only vary between 0 and +1. A correlation between gametes as calculated from a pedigree cannot be negative ... It may, however, be useful tofind the correlation between such gametes relative to the array of gametes of their own generation. Such a correlation (not capable of direct calculation from the pedigree) may be negative. Thus it is negative by definition, if there is maximum avoidance of consanguinity ..."
